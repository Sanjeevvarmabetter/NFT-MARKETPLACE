## NFT Marketplace  (Polygon Amoy Testnet)
This project is a decentralized NFT marketplace built on the Polygon Mumbai testnet, using Hardhat for smart contract development and React for the frontend. It allows users to mint, buy, and sell NFTs with low transaction fees. The marketplace leverages Polygon’s scalability, providing a seamless user experience with MetaMask integration.

### Key Features:
-Mint, buy, and sell NFTs.
-Low gas fees using Polygon.
-React-based frontend with MetaMask wallet support.
-Smart contracts built with Solidity and deployed via Hardhat.
